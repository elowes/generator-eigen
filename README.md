[Issues](https://github.com/elowes/generator-eigen/issues )Issue 提交

## Installation

1. 全局安装 yeoman
``` shell
npm install yo -g
```

2. 安装 generator-eigen
``` shell
npm install generator-eigen
```

3. 在本地新建项目文件夹，例如文件夹 tmp
``` shell
mkdir tmp && cd tmp
```

4. 执行命令生成项目模板（基于 react@16.0 & dva@2.0）
``` shell
yo eigen
```

5. 启动开发服务器
``` shell
npm start
```

6. 打包代码 (Only Support MacOS)
``` shell
npm run build
```
